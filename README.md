# linalg

A Simple Linear Algebra Package

## Getting Started

```dart
final Matrix a = Matrix([[1, 2], [3, 4]]);
final Matrix b = Matrix([[2, 0], [1, 2]]);
final Matrix e = Matrix([[4, 4], [10, 8]]);
Matrix result = a * b
print(result)
print(result == e)
```
